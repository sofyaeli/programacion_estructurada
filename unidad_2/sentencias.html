Unidad 2: Estructuras de Control

Objetivo

Comprender el funcionamiento de las estructuras de control condicionales en C#, específicamente las sentencias if y switch, para tomar decisiones dentro de un programa según las condiciones establecidas por el usuario o por la lógica de la aplicación.

2.1 La Sentencia de Control IF

La sentencia if permite ejecutar un bloque de código únicamente cuando una condición es verdadera. Es una de las estructuras más utilizadas en programación para la toma de decisiones.

Sintaxis

if (condicion)
{
    // Instrucciones
}

Cuando la condición se evalúa como true, las instrucciones dentro del bloque se ejecutan. Si la condición es falsa, el bloque es ignorado.

Ejemplo Simple

int edad = 20;

if (edad >= 18)
{
    Console.WriteLine("Es mayor de edad");
}

En este ejemplo, el mensaje se mostrará porque la condición edad >= 18 es verdadera.

Uso de IF - ELSE

La estructura if - else permite ejecutar un bloque de instrucciones cuando la condición es verdadera y otro bloque diferente cuando es falsa. 

int edad = 16;

if (edad >= 18)
{
    Console.WriteLine("Mayor de edad");
}
else
{
    Console.WriteLine("Menor de edad");
}

Ventajas del IF

  • Permite tomar decisiones dentro del programa.
  • Facilita la validación de datos.
  • Ayuda a controlar el flujo de ejecución.
  • Es fácil de implementar y comprender.

2.2 La Sentencia de Control SWITCH

La sentencia switch permite seleccionar una acción entre varias alternativas posibles según el valor de una variable o expresión.

Sintaxis

switch(variable)
{
    case valor1:
        // Instrucciones
        break;

    case valor2:
        // Instrucciones
        break;

    default:
        // Instrucciones por defecto
        break;
}

Cada opción se representa mediante la palabra reservada case. Si ninguna condición coincide, se ejecuta el bloque default.

Ejemplo

int opcion = 2;

switch(opcion)
{
    case 1:
        Console.WriteLine("Opción 1");
        break;

    case 2:
        Console.WriteLine("Opción 2");
        break;

    case 3:
        Console.WriteLine("Opción 3");
        break;

    default:
        Console.WriteLine("Opción inválida");
        break;
}

En este caso se mostrará el mensaje "Opción 2" porque el valor de la variable coincide con el segundo caso.

Diferencias entre IF y SWITCH

  • if es ideal para evaluar condiciones complejas utilizando operadores relacionales y lógicos.
  • switch es más adecuado cuando se desea seleccionar entre varios valores específicos.
  • El uso de switch puede hacer el código más organizado cuando existen muchas opciones.

Aplicaciones Prácticas

  • Sistemas de menús.
  • Calculadoras.
  • Validación de usuarios.
  • Control de acceso.
  • Selección de opciones en formularios.

Conclusiones

  • La sentencia if permite ejecutar instrucciones dependiendo de una condición lógica.
  • La estructura switch facilita la selección entre múltiples alternativas de forma organizada.
  • Ambas estructuras son fundamentales para la toma de decisiones en cualquier programa desarrollado en C#.
  • El dominio de estas sentencias permite desarrollar aplicaciones más dinámicas, eficientes y adaptables a diferentes situaciones.

Trabajo Autónomo No. 03

Objetivos

Desarrollar un programa en C# que permita determinar si un número entero ingresado por el usuario es múltiplo de 5 utilizando el operador módulo (%). Mediante esta actividad se busca comprender el uso de operadores aritméticos y estructuras condicionales como if, fortaleciendo la capacidad de análisis lógico para la resolución de problemas y promoviendo el compromiso ético en el desarrollo de soluciones informáticas.

Descripción del Problema

Se requiere elaborar un programa que solicite al usuario un número entero y determine si dicho número es múltiplo de 5. Para ello se utilizará el operador módulo (%), que permite obtener el residuo de una división.

Un número es múltiplo de 5 cuando el residuo de dividirlo para 5 es igual a cero.

Fundamento Teórico

El operador % devuelve el residuo de una división entre dos números enteros. 

15 % 5 = 0
17 % 5 = 2
20 % 5 = 0

Si el resultado de numero % 5 es igual a cero, significa que el número es múltiplo de 5.

Algoritmo

  • Inicio.
  • Solicitar al usuario un número entero.
  • Calcular el residuo utilizando el operador %.
  • Verificar si el residuo es igual a cero.
  • Si es verdadero, indicar que el número es múltiplo de 5.
  • Si es falso, indicar que el número no es múltiplo de 5.
  • Fin.

Código en C#

using System;

class Program
{
    static void Main()
    {
        int numero;

        Console.Write("Ingrese un número entero: ");
        numero = int.Parse(Console.ReadLine());

        if (numero % 5 == 0)
        {
            Console.WriteLine(
                "El número es múltiplo de 5."
            );
        }
        else
        {
            Console.WriteLine(
                "El número no es múltiplo de 5."
            );
        }
    }
}

Ejemplo de Ejecución

  • Número ingresado: 25
  • Operación realizada: 25 % 5 = 0
  • Resultado: El número es múltiplo de 5.

Análisis de Resultados

Cuando el usuario ingresa el número 25, el programa realiza la operación 25 % 5, obteniendo como resultado 0. Debido a que el residuo es igual a cero, la condición del if se cumple y el sistema informa que el número es múltiplo de 5.

Trabajo Autónomo No. 04

Objetivos

Desarrollar programas en C# que permitan calcular el factorial de un número entero positivo ingresado por el usuario utilizando dos estructuras repetitivas diferentes: while y for. Mediante esta actividad se busca comprender el funcionamiento de los ciclos, comparar sus implementaciones y fortalecer el pensamiento crítico y reflexivo para la resolución de problemas mediante programación.

Descripción del Problema

Se requiere elaborar un programa que solicite un número entero positivo N y calcule su factorial. Además, se deben desarrollar dos versiones de la solución: una utilizando la sentencia while y otra empleando la sentencia for, con el propósito de comparar ambas estructuras de repetición.

Fundamento Teórico

El factorial de un número entero positivo N se representa mediante el símbolo N! y consiste en multiplicar todos los números enteros desde 1 hasta N.

Por ejemplo:

5! = 5 × 4 × 3 × 2 × 1 = 120

4! = 4 × 3 × 2 × 1 = 24

3! = 3 × 2 × 1 = 6

El factorial es ampliamente utilizado en matemáticas, estadística, combinatoria e informática.

Algoritmo

  • Inicio.
  • Solicitar un número entero positivo N.
  • Inicializar la variable factorial en 1.
  • Multiplicar factorial por cada número desde 1 hasta N.
  • Mostrar el resultado obtenido.
  • Fin.

Implementación con WHILE

using System;

class Program
{
    static void Main()
    {
        int numero, contador = 1;
        long factorial = 1;

        Console.Write("Ingrese un número: ");
        numero = int.Parse(Console.ReadLine());

        while (contador <= numero)
        {
            factorial *= contador;
            contador++;
        }

        Console.WriteLine(
            "El factorial es: " + factorial
        );
    }
}

Implementación con FOR

using System;

class Program
{
    static void Main()
    {
        int numero;
        long factorial = 1;

        Console.Write("Ingrese un número: ");
        numero = int.Parse(Console.ReadLine());

        for (int i = 1; i <= numero; i++)
        {
            factorial *= i;
        }

        Console.WriteLine(
            "El factorial es: " + factorial
        );
    }
}

Ejemplo de Ejecución

  • Número ingresado: 5
  • Cálculo: 5 × 4 × 3 × 2 × 1
  • Resultado: 120

Comparación entre WHILE y FOR

  • La estructura while ejecuta las instrucciones mientras la condición sea verdadera.
  • La estructura for integra la inicialización, condición e incremento en una sola línea.
  • Para recorridos con una cantidad conocida de iteraciones, el ciclo for suele ser más compacto y legible.
  • El ciclo while resulta útil cuando no se conoce previamente el número exacto de repeticiones.

Análisis de Resultados

Ambas implementaciones producen exactamente el mismo resultado. La diferencia principal radica en la forma de controlar las iteraciones. Mientras que el ciclo while requiere una variable de control externa, el ciclo for permite gestionar todo el proceso en una sola estructura.

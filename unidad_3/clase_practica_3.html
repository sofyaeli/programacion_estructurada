Clase Práctica No. 3

Objetivos

Desarrollar una aplicación en C# para la gestión de un videoclub que permita administrar películas y videojuegos disponibles para alquiler. Mediante esta práctica se busca aplicar los conceptos de Programación Orientada a Objetos, utilizando clases, objetos, propiedades, métodos y estructuras de control para gestionar el inventario, controlar la disponibilidad de los artículos y registrar las operaciones de alquiler y devolución. Además, esta actividad fomenta el trabajo en equipo y la resolución de problemas mediante el diseño de sistemas informáticos.

Descripción del Problema

Se requiere desarrollar un sistema para un videoclub que alquila películas y videojuegos. El programa deberá registrar todos los artículos disponibles, controlar cuáles se encuentran alquilados y llevar un conteo del número de veces que cada uno ha sido rentado.

El sistema deberá permitir gestionar dos tipos de elementos:

  • Películas.
  • Videojuegos.

Características de los Videojuegos

  • Nombre.
  • Año de edición.
  • Sistema (PlayStation, PS2, Xbox, PC, etc.).
  • Descripción.
  • Número de registro único.

Características de las Películas

  • Título.
  • Clase o género.
  • País de producción.
  • Número de registro único.

Funciones del Sistema

  • Registrar películas.
  • Registrar videojuegos.
  • Generar automáticamente números de registro.
  • Mostrar todos los artículos registrados.
  • Mostrar artículos alquilados.
  • Mostrar artículos disponibles.
  • Consultar una película por título.
  • Consultar un videojuego por nombre.
  • Realizar alquileres.
  • Registrar devoluciones.
  • Contabilizar el número de alquileres realizados.

Diseño de Clases

Clase Material

Representa las características comunes entre películas y videojuegos. 

class Material
{
    public int NumeroRegistro { get; set; }

    public bool Alquilado { get; set; }

    public int VecesAlquilado { get; set; }
}

Clase Pelicula

class Pelicula : Material
{
    public string Titulo { get; set; }

    public string Genero { get; set; }

    public string PaisProduccion { get; set; }
}

Clase Videojuego

class Videojuego : Material
{
    public string Nombre { get; set; }

    public int AnioEdicion { get; set; }

    public string Sistema { get; set; }

    public string Descripcion { get; set; }
}

Generación Automática del Número de Registro

Cada vez que se registre una película o videojuego, el sistema asignará automáticamente un identificador único. 

private static int contador = 1;

NumeroRegistro = contador++;

Esto garantiza que no existan dos artículos con el mismo número de registro.

Proceso de Alquiler

  • Ingresar el número de registro.
  • Verificar disponibilidad.
  • Si está disponible, marcar como alquilado.
  • Incrementar el contador de alquileres.
  • Mostrar mensaje de confirmación.
if (!material.Alquilado)
{
    material.Alquilado = true;
    material.VecesAlquilado++;
}

Proceso de Devolución

  • Ingresar número de registro.
  • Buscar el artículo.
  • Actualizar estado a disponible.
material.Alquilado = false;

La implementación de un sistema de videoclub mediante Programación Orientada a Objetos permite organizar mejor la información y reutilizar código.

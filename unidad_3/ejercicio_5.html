Trabajo Autónomo No. 05

Objetivos

Desarrollar una aplicación en C# que permita realizar operaciones con números complejos mediante la implementación de clases y métodos propios de la Programación Orientada a Objetos. A través de esta actividad se busca comprender la creación de clases, la interacción entre objetos y la implementación de operaciones matemáticas como suma, resta, multiplicación, división y conversiones entre forma binomial y polar. Además, se fomenta el compromiso ético en el desarrollo de soluciones informáticas confiables y estructuradas.

Descripción del Problema

Se requiere desarrollar un programa para operar con números complejos. El sistema estará conformado por tres clases:

  • Clase Principal: contiene el método Main.
  • Clase NumeroComplejo: representa un número complejo.
  • Clase AritmeticaCompleja: contiene los métodos para realizar operaciones entre números complejos.

El programa deberá realizar las siguientes operaciones:

  • Suma de números complejos.
  • Resta de números complejos.
  • Multiplicación de números complejos.
  • División de números complejos.
  • Conversión de forma binomial a polar.
  • Conversión de forma polar a binomial.

Fundamento Teórico

Un número complejo está formado por una parte real y una parte imaginaria: 

z = a + bi

Donde:

  • a = parte real
  • b = parte imaginaria
  • i = √(-1)

Los números complejos pueden expresarse en:

  • Forma Binomial: a + bi
  • Forma Polar: r(cos θ + i sen θ)

Diseño de Clases

Clase NumeroComplejo

class NumeroComplejo
{
    public double Real { get; set; }
    public double Imaginario { get; set; }

    public NumeroComplejo(
        double real,
        double imaginario)
    {
        Real = real;
        Imaginario = imaginario;
    }
}

Clase AritmeticaCompleja

class AritmeticaCompleja
{
    public NumeroComplejo Sumar(
        NumeroComplejo a,
        NumeroComplejo b)
    {
        return new NumeroComplejo(
            a.Real + b.Real,
            a.Imaginario + b.Imaginario);
    }
}

Algoritmo

  • Inicio.
  • Crear objetos de tipo NumeroComplejo.
  • Ingresar la parte real e imaginaria.
  • Seleccionar la operación deseada.
  • Ejecutar el método correspondiente.
  • Mostrar el resultado obtenido.
  • Fin.

Código Principal

using System;

class Program
{
    static void Main()
    {
        NumeroComplejo n1 =
            new NumeroComplejo(4, 3);

        NumeroComplejo n2 =
            new NumeroComplejo(2, 1);

        AritmeticaCompleja op =
            new AritmeticaCompleja();

        NumeroComplejo resultado =
            op.Sumar(n1, n2);

        Console.WriteLine(
            resultado.Real + " + " +
            resultado.Imaginario + "i");
    }
}

Ejemplo de Ejecución

  • Primer complejo: 4 + 3i
  • Segundo complejo: 2 + 1i
  • Operación: Suma
  • Resultado: 6 + 4i

Conversión de Forma Binomial a Polar

Para convertir un número complejo de forma binomial a polar se utilizan las siguientes expresiones: 

r = √(a² + b²)

θ = atan(b / a)

Donde:

  • r = módulo
  • θ = argumento

Conversión de Forma Polar a Binomial

a = r cos(θ)

b = r sen(θ)

Estas fórmulas permiten obtener nuevamente la parte real e imaginaria del número complejo.

Ejercicio 2

Objetivos

Desarrollar un programa en C# que permita calcular las raíces de una ecuación de segundo grado utilizando el método Math.Sqrt() del espacio de nombres System. Mediante este ejercicio se busca comprender los conceptos fundamentales de las ecuaciones cuadráticas, el uso de operaciones matemáticas avanzadas en programación y la aplicación de estos conocimientos para resolver problemas de manera lógica y responsable, fortaleciendo además el compromiso ético en el desarrollo de soluciones informáticas.

Descripción del Problema

Una ecuación de segundo grado tiene la forma:

ax² + bx + c = 0

Donde a, b y c son coeficientes proporcionados por el usuario. Para encontrar las raíces se utiliza la fórmula general: 

x1 = (-b + √(b² - 4ac)) / (2a)

x2 = (-b - √(b² - 4ac)) / (2a)

El valor dentro de la raíz cuadrada se denomina discriminante:

D = b² - 4ac

Si D es mayor o igual a cero, existen raíces reales. En caso contrario, las raíces son números complejos y no podrán calcularse mediante Math.Sqrt().

Algoritmo

  • Inicio.
  • Solicitar los coeficientes a, b y c.
  • Calcular el discriminante D = b² - 4ac.
  • Verificar si D es mayor o igual a cero.
  • Si es verdadero, calcular las dos raíces utilizando la fórmula general.
  • Mostrar las raíces obtenidas.
  • Si D es negativo, informar que no existen raíces reales.
  • Fin.

Código en C#

using System;

class Program
{
    static void Main()
    {
        double a, b, c;
        double discriminante, x1, x2;

        Console.Write("Ingrese el valor de a: ");
        a = double.Parse(Console.ReadLine());

        Console.Write("Ingrese el valor de b: ");
        b = double.Parse(Console.ReadLine());

        Console.Write("Ingrese el valor de c: ");
        c = double.Parse(Console.ReadLine());

        discriminante = (b * b) - (4 * a * c);

        if (discriminante >= 0)
        {
            x1 = (-b + Math.Sqrt(discriminante)) / (2 * a);
            x2 = (-b - Math.Sqrt(discriminante)) / (2 * a);

            Console.WriteLine("\nRaíz 1: " + x1);
            Console.WriteLine("Raíz 2: " + x2);
        }
        else
        {
            Console.WriteLine(
                "\nLa ecuación no posee raíces reales."
            );
        }
    }
}

Ejemplo de Ejecución

  • a = 1
  • b = -5
  • c = 6
  • Discriminante = 1
  • Raíz 1: 3
  • Raíz 2: 2

Análisis de Resultados

En el ejemplo anterior se obtuvo un discriminante positivo, lo que indica que la ecuación posee dos raíces reales diferentes. El método Math.Sqrt() permitió calcular la raíz cuadrada del discriminante y obtener los valores que satisfacen la ecuación cuadrática.

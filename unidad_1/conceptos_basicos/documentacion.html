Conceptos Básicos de C#

Objetivo

Conocer los fundamentos del lenguaje de programación C#, incluyendo los tipos de datos básicos, declaración de variables, entrada de datos, operadores, palabras reservadas, variables y constantes.

1.1 Tipos de datos básicos, declaración de variables y entrada de datos

Los tipos de datos permiten almacenar información de diferentes clases dentro de un programa.

  • int: almacena números enteros.
  • double: almacena números decimales.
  • float: almacena números decimales de precisión simple.
  • char: almacena un único carácter.
  • string: almacena texto.
  • bool: almacena valores verdadero o falso.

Ejemplo de declaración de variables:

int edad = 20;
double salario = 850.50;
char letra = 'A';
string nombre = "Sofia";
bool activo = true;

Entrada de datos:

Console.Write("Ingrese su nombre: ");
string nombre = Console.ReadLine();

Console.Write("Ingrese su edad: ");
int edad = int.Parse(Console.ReadLine());

1.2 Operadores aritméticos, relacionales y lógicos

Los operadores permiten realizar cálculos, comparaciones y evaluar condiciones dentro de un programa.

Operadores Aritméticos

  • + : Suma
  • - : Resta
  • * : Multiplicación
  • / : División
  • % : Módulo o residuo
int a = 10;
int b = 3;

Console.WriteLine(a + b);
Console.WriteLine(a - b);
Console.WriteLine(a * b);
Console.WriteLine(a / b);
Console.WriteLine(a % b);

Operadores Relacionales

  • == Igual a
  • != Diferente de
  • > Mayor que
  • < Menor que
  • >= Mayor o igual que
  • <= Menor o igual que
int edad = 18;
Console.WriteLine(edad >= 18);

Operadores Lógicos

  • && : AND
  • || : OR
  • ! : NOT
bool tieneLicencia = true;
bool esMayor = true;

Console.WriteLine(tieneLicencia && esMayor);

1.3 Palabras reservadas

Son palabras que tienen un significado especial para el compilador y no pueden utilizarse como nombres de variables.

  • int
  • double
  • string
  • bool
  • if
  • else
  • for
  • while
  • return
  • class
  • public
  • private
  • static
  • void
int class = 10; // Error porque class es reservada

1.4 Variables y Constantes en C#

Las variables almacenan información que puede cambiar durante la ejecución del programa. 

int edad = 20;
edad = 21;

Las constantes almacenan valores fijos que no pueden modificarse una vez declarados. 

const double PI = 3.1416;

Ejemplo:

const double IVA = 0.15;

double precio = 100;
double total = precio + (precio * IVA);

Console.WriteLine(total);

Conclusiones

  • Los tipos de datos permiten almacenar información de acuerdo con su naturaleza, ya sean números, texto o valores lógicos.
  • Los operadores facilitan la realización de cálculos y la evaluación de condiciones dentro de un programa.
  • Las palabras reservadas forman parte de la sintaxis propia de C# y no pueden utilizarse como identificadores.
  • Las variables permiten almacenar datos modificables, mientras que las constantes garantizan valores fijos durante toda la ejecución del programa.

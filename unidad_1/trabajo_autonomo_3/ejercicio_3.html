Clase Práctica No. 1

Objetivos

Diseñar e implementar una calculadora simple en C# capaz de realizar operaciones aritméticas básicas entre dos números enteros. Mediante esta práctica se busca comprender el uso de variables, operadores aritméticos, entrada y salida de datos, así como la utilización de estructuras condicionales para tomar decisiones según la operación seleccionada por el usuario. Esta actividad contribuye al desarrollo del pensamiento crítico y reflexivo para la resolución de problemas mediante programación.

Descripción del Problema

Se requiere desarrollar un programa que funcione como una calculadora básica. El sistema deberá solicitar al usuario dos números enteros y una operación aritmética (+, -, * o /). Posteriormente, el programa evaluará la operación ingresada y mostrará el resultado correspondiente.

Operaciones Disponibles

  • + Suma
  • - Resta
  • * Multiplicación
  • / División

Algoritmo

  • Inicio.
  • Solicitar el primer número entero.
  • Solicitar el segundo número entero.
  • Solicitar la operación aritmética.
  • Evaluar la operación ingresada.
  • Realizar el cálculo correspondiente.
  • Mostrar el resultado.
  • Si la operación no es válida, mostrar un mensaje de error.
  • Fin.

Código en C#

using System;

class Program
{
    static void Main()
    {
        int num1, num2;
        char operacion;

        Console.Write("Ingrese el primer número: ");
        num1 = int.Parse(Console.ReadLine());

        Console.Write("Ingrese el segundo número: ");
        num2 = int.Parse(Console.ReadLine());

        Console.Write("Ingrese la operación (+, -, *, /): ");
        operacion = char.Parse(Console.ReadLine());

        switch (operacion)
        {
            case '+':
                Console.WriteLine(
                    "Resultado: " + (num1 + num2)
                );
                break;

            case '-':
                Console.WriteLine(
                    "Resultado: " + (num1 - num2)
                );
                break;

            case '*':
                Console.WriteLine(
                    "Resultado: " + (num1 * num2)
                );
                break;

            case '/':
                if (num2 != 0)
                {
                    Console.WriteLine(
                        "Resultado: " +
                        (double)num1 / num2
                    );
                }
                else
                {
                    Console.WriteLine(
                        "Error: No se puede dividir para cero."
                    );
                }
                break;

            default:
                Console.WriteLine(
                    "Operación no válida."
                );
                break;
        }
    }
}

Ejemplo de Ejecución

  • Primer número: 15
  • Segundo número: 5
  • Operación: +
  • Resultado: 20

Análisis de Resultados

El programa identifica la operación ingresada por el usuario y ejecuta el cálculo correspondiente mediante una estructura condicional switch. Esto permite seleccionar la acción adecuada de forma eficiente y mostrar el resultado esperado.

Conceptos Aplicados

  • Variables y constantes.
  • Operadores aritméticos.
  • Entrada y salida de datos.
  • Estructuras de control condicionales.
  • Validación de datos para evitar errores.

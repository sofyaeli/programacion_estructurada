Ejercicio 1

Objetivos

Desarrollar un programa en C# que solicite al usuario tres valores enteros, calcule su suma y su media aritmética. Mediante este ejercicio se busca comprender el uso de variables, entrada de datos, operadores aritméticos y salida de información, fortaleciendo el pensamiento lógico y la capacidad para resolver problemas cotidianos mediante la programación.

Descripción del Problema

Se requiere elaborar un programa que permita ingresar tres números enteros proporcionados por el usuario. Posteriormente, el programa deberá calcular la suma de los tres valores y determinar su media aritmética (promedio), mostrando los resultados obtenidos en pantalla.

Algoritmo

  • Inicio.
  • Solicitar al usuario el primer número entero.
  • Solicitar al usuario el segundo número entero.
  • Solicitar al usuario el tercer número entero.
  • Calcular la suma de los tres números.
  • Calcular la media dividiendo la suma para tres.
  • Mostrar la suma obtenida.
  • Mostrar la media obtenida.
  • Fin.

Código en C#

using System;

class Program
{
    static void Main()
    {
        int numero1, numero2, numero3;
        int suma;
        double media;

        Console.Write("Ingrese el primer número: ");
        numero1 = int.Parse(Console.ReadLine());

        Console.Write("Ingrese el segundo número: ");
        numero2 = int.Parse(Console.ReadLine());

        Console.Write("Ingrese el tercer número: ");
        numero3 = int.Parse(Console.ReadLine());

        suma = numero1 + numero2 + numero3;
        media = suma / 3.0;

        Console.WriteLine("\nLa suma es: " + suma);
        Console.WriteLine("La media es: " + media);
    }
}

Ejemplo de Ejecución

  • Primer número: 10
  • Segundo número: 20
  • Tercer número: 30
  • Suma: 60
  • Media: 20

Conclusiones

  • Se aplicaron conceptos fundamentales de programación como variables, entrada y salida de datos.
  • Se utilizó el operador de suma para obtener el total de los tres números ingresados.
  • Se calculó la media aritmética utilizando operaciones matemáticas básicas.
  • Este ejercicio demuestra cómo la programación puede emplearse para resolver problemas sencillos de la vida cotidiana de manera rápida y eficiente.
